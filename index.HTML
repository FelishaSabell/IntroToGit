Intro to Git & Github
Command description 
git init
 Initializes our repo[ONLY RUN ONCE, when the repo is first made] 
git status
 displays status of files being tracked 
git add
 stages any changes files that you tell it to 
git commit -m "your message"
 clears the staging, saves the changes on your LOCAL repo/machine, and provides a "why" 
git push
 upload your commits to a remote repository 
git remote add name "link"
 Add a link to a remote repository that we can share information with
